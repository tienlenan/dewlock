# 14 — Product Naming

Status: **Working name = `Dewlock`** (recommended). Runner-up = `Dewpoint`. ⚠️ NOT yet legally verified — see Verification checklist before committing. Source: 2 naming workflows (generate → vet collisions/availability → judge panel → shortlist), 2026-06-15.

Brand direction (user choice): tie to **Sui's identity — the water-droplet logo, pastel/aqua blue** ("Sui" = water 水). Theme = droplet/aqua/calm, but the product is **security-serious** → water is the noun, security is the verb.

## Recommendation

**SHIP: `Dewlock`** — the only candidate that encodes BOTH brand pillars in one word: **dew** = the pastel Sui droplet; **lock** = the fail-closed Guardian that BLOCKS dangerous txs. Coined compound → strong/ownable trademark. Top judge score (52). Confirmed clear in crypto/DeFi/Sui namespace (only `dewlock.com` = an unrelated IT consultancy, different class). Tagline: *Every transaction, sealed before you sign.*
Caveat: `-lock` lightly grazes the guardian/`-ward` lane (Warden $WARD) — defensible ("lock" is a universal primitive, "dew-" anchors it in water).

**RUNNER-UP: `Dewpoint`** — same dew/droplet logo family (one-swap fallback), "point" adds precision (un-soft), crypto-clear. Use if legal rules `-lock` too close to the ward lane. Downside: generic English compound → weaker trademark/SEO, lives on `.io/.xyz`. Tagline: *The moment intent becomes safe to sign.*

## Logo / brand direction (counters the "soft" risk)

Water = approachable on-ramp; the lock/precision layer = the credibility that earns the signature.
- **Mark:** pastel-aqua dew bead whose lower curve morphs into a keyhole / clicks into a rounded padlock; soft surface-tension highlight.
- **Type:** confident, slightly technical geometric grotesk (fintech-precise), NOT rounded/script.
- **Palette:** Sui pastel-blue for warmth + a deep-blue / dark-ink accent for gravitas.
- **Copy:** lead with active guardian verbs — sealed / blocked / verified / re-derived / safe to sign. Water is the noun; security is the verb.

## Avoid (verified collisions / landmines)

- **Levee** → Levva/LVVA is a live, exchange-listed "AI Copilot" DeFi portfolio manager — same category + near-homophone.
- **Veto** → "Veto" = an authorization kernel for AI agents (block/allow/escalate) — identical category.
- **Quell / Suika / Mizuna** → collisions.
- **`-ward` lane** (Brineward/Tuskward/Wardstone) → Warden Protocol $WARD (agentic copilot, self-custody) — derivative risk.
- **`Sui-` prefix** (Suiren etc.) → implies official Sui Foundation affiliation + "Sui" is a protected mark; risky for a trust tool.
- Pure-cute water words (Dewdrop, Mizuna) → read like a spa/wellness app, undercut the security headline.
