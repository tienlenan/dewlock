Dewlock

Every transaction, sealed before you sign.

Sui Overflow 2026 · DeFi & Payments + DeepBook · Live on Sui mainnet → dewlock.vercel.app

An intent-firewall for agentic Sui DeFi. State a goal in plain language → an AI agent (holding zero user-fund keys) compiles it into one unsigned transaction → a deterministic, fail-closed Guardian re-derives the math and dry-runs that exact bundle → you sign the literal artifact you reviewed (WYSIWYS) → an immutable receipt lands on Walrus.

Why it's different

The LLM only proposes; trust lives in deterministic Guardian code, not the model. The server never holds your keys, you sign exactly the bytes you previewed, token identity is by on-chain coin type (not spoofable symbol), and a refused action writes a provable on-chain BLOCK receipt.

How it works

intent → agent proposes 1 unsigned PTB → GUARDIAN (11 fail-closed gates)
   ├─ BLOCK → immutable proof receipt
   └─ PASS  → WYSIWYS sign → Walrus receipt + Sui anchor

Features

  • Conversational copilot with generative UI cards (portfolio, tx preview, swap picker, receipts).
  • Swap (Cetus Aggregator + Aftermath, source-aware min-out), transfer (SuiNS + friend book), lend (NAVI/Suilend), DeepBook POST_ONLY limit orders, Wormhole redeem.
  • Fail-closed Guardian — allowlist, WYSIWYS digest, trusted-price caps, min-out re-derive, price-impact / slippage guard, native-SUI gas safety.
  • Seal-encrypted conversations — personal chat encrypted client-side; only your wallet decrypts.
  • Immutable receipts + a Passport (level / XP / badges) on Walrus & Sui.
