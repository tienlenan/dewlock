How it works

Four steps. You sign once.

The agent does the heavy lifting; your wallet is the only key that matters.

01
State your intent

Tell Dewlock what you want in plain language: "swap 10 SUI to USDC" or "limit buy DEEP at 0.0031".

02
Agent builds one PTB

One unsigned programmable transaction block — the complete atomic payload, built without touching your keys.

03
Guardian re-derives + dry-runs

A deterministic fail-closed Guardian re-checks the math and dry-runs on-chain. Any mismatch → hard block.

04
You sign. Walrus logs.

Your wallet receives the exact PTB you reviewed. On success, an immutable Walrus receipt anchored to a Sui object records it.

Security model

Fail-closed. Not trust-me.

The Guardian re-derives every critical value independently — never relying on what the agent computed. Any ambiguity, simulation failure, or parse error closes the gate.

Zero user keys server-side

The agent builds unsigned PTBs only. Your private key never leaves your wallet.

Min-out re-derivation

Re-fetches the live Cetus quote and re-calculates minimum received from scratch. Diverges past slippage → block.

Coin-type gate, not symbol

Assets identified by full on-chain Move type (0x2::sui::SUI), not ticker. Spoofed USDC → blocked.

SuiNS lookalike guard

Resolved 0x is pinned at parse time. A lookalike .sui resolving elsewhere is a hard block before the PTB is built.

Deterministic fail-closed

Simulation failure, parse error, RPC timeout → the Guardian blocks, not degrades. No silent fallbacks.

Allowlist + BalanceManager cap

Only whitelisted contracts can be called. A hard cap limits per-trade and daily spend.

Built on Sui

The stack that makes this possible.

Dewlock isn't a thin wrapper — it relies on four Sui-native capabilities that have no direct equivalent elsewhere.

DeFi & Payments
Cetus AMM · SuiNS · PTB composition

Atomic multi-protocol bundles. SuiNS name resolution with anti-lookalike guard. Min-out re-derivation against the Cetus math library — the same failure class behind the Cetus exploit.

DeepBook
@mysten/deepbook-v3 · BalanceManager

Real central limit order book on Sui mainnet. POST_ONLY limit orders. Hard spending cap via BalanceManager. The only venue where "wait at my price" is a native primitive.

Walrus
Walrus blob storage · Sui object anchor

Content-addressed immutable receipts for every action — executed or blocked. The blob is the proof; the Sui object is the discoverable pointer.

Agentic Web
Mastra agent · unsigned PTBs only

Natural language to on-chain action with human-in-the-loop signing. The agent builds; you authorize. Zero agent custody. Demonstrably keyless server-side.
